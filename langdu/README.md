# 📖 朗读文档小软件

上传 Word（.docx）、TXT 或 Markdown（.md）文档，选择喜欢的普通话声音，
用自然流畅、富有感情的声音为你朗读。

语音引擎使用微软 Edge 的神经网络语音（edge-tts），**完全免费**，
朗读效果接近真人，不是机械音。

## 功能

- 支持 .docx、.txt、.md 三种文档格式（TXT 自动识别 UTF-8 / GBK 编码）
- **6 种普通话声音**：女声晓晓、晓伊；男声云希、云健、云扬（新闻播报）、云夏（讲故事）
- 可调语速（慢 50% 到 快 50%）
- **长文档自动分段朗读**：一段一段读，衔接顺畅，不用一次等很久
- **暂停 / 继续 / 停止**：随时控制朗读
- **进度显示**：右下方列出所有段落，正在读的那段会高亮，点任意一段可以从那里开始听
- 上传后可以先预览、修改文字，再朗读

## 怎么运行（macOS）

### 做成桌面 App 图标（最方便，推荐）

双击运行一次 **「制作App图标.command」**，它会在你的**桌面**上生成一个
独立的「**朗读文档**」App 图标。之后：

- 双击桌面那个图标即可打开，**全程不出现终端**；关掉窗口就退出。
- 可以把图标拖到 Dock 栏常驻。

> 第一次双击 .command 若被系统拦截，**右键点它 → 选「打开」** 即可。

### 双击启动（依赖终端的版本）

直接双击 **「启动朗读软件.command」**，会先弹出终端、再打开应用窗口。
（这个黑色终端关掉的话，App 也会一起关闭。）

### 手机 / 平板也能用（同一个 Wi-Fi）

双击 **「手机也能用.command」**，电脑屏幕上会显示一个网址（形如
`http://192.168.x.x:8000`）。在手机浏览器里打开这个网址，就能用手机
操作朗读软件。

要求：手机和电脑连**同一个 Wi-Fi**；电脑要开着、别关那个窗口；第一次
系统若问「是否允许接入网络连接」请点「允许」。

> 说明：手机本身不运行程序，真正干活的是电脑；手机相当于一块“遥控屏幕”。
> 因此电脑必须开着并保持运行。

### 或者：用终端

按 `Command + 空格` 搜索「终端」打开，进入这个文件夹后执行：

```bash
pip3 install -r requirements.txt
python3 app_gui.py        # 打开独立应用窗口（推荐）
```

如果想用浏览器版（网页版），改成 `python3 app.py`，再用浏览器打开 http://127.0.0.1:8000 。

### 装成手机/电脑上的图标（PWA）

本软件已做成 PWA，可以“添加到主屏幕”，全屏打开、像一个独立 App。

- **手机上（同一 Wi-Fi 先体验）**：先在电脑双击「手机也能用.command」，
  手机浏览器打开提示的网址 `http://192.168.x.x:8000` → iPhone 用 Safari
  点底部「分享」→「添加到主屏幕」；安卓用 Chrome 在菜单里选「安装应用/添加到主屏幕」。
  之后点主屏幕图标即可全屏打开。
- **电脑上**：用 Chrome / Edge 打开 `http://127.0.0.1:8000`，地址栏右侧会出现
  「安装」图标，点一下即可装成桌面应用。

> 说明：
> - 局域网用的是普通 http，iPhone 仍可“添加到主屏幕”并全屏打开；离线缓存
>   （Service Worker）只在 https 或电脑本机(localhost)下启用，不影响日常使用。
> - 等将来部署到带 https 的服务器（如阿里云），离线缓存、二次打开更快等
>   完整 PWA 能力会自动启用，而且在外面用流量也能开。
> - iPhone 上锁屏/后台朗读支持有限，建议看着屏幕用。

## 用完之后

回到终端窗口，按 `Control + C` 就能关闭软件。

## 注意

- 朗读时需要联网（语音是微软的在线服务生成的）
- 文档越长，第一段生成语音会等几秒，之后会自动提前准备下一段，听起来更连贯
- 老版 Word 的 .doc 格式不支持，请先用 Word 另存为 .docx
