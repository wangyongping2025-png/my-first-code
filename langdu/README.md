# 📖 朗读文档小软件

上传 Word（.docx）、TXT 或 Markdown（.md）文档，选择男声或女声，
用自然流畅、富有感情的普通话为你朗读，还可以把语音下载成 MP3。

语音引擎使用微软 Edge 的神经网络语音（edge-tts），**完全免费**，
朗读效果接近真人，不是机械音。

## 功能

- 支持 .docx、.txt、.md 三种文档格式（TXT 自动识别 UTF-8 / GBK 编码）
- 女声「晓晓」（温暖亲切）和男声「云希」（沉稳自然）两种声音
- 可调语速（慢 50% 到 快 50%）
- 上传后可以先预览、修改文字，再朗读
- 朗读结果可以直接播放，也可以下载成 MP3 文件

## 怎么运行

需要先安装好 Python 3（3.9 以上版本）。在这个文件夹里打开终端，执行：

```bash
pip install -r requirements.txt
python app.py
```

看到提示后，用浏览器打开 http://127.0.0.1:5000 就可以使用了。

## 注意

- 朗读时需要联网（语音是微软的在线服务生成的）
- 文档越长，生成语音等待的时间越久，请耐心等待
- 老版 Word 的 .doc 格式不支持，请先用 Word 另存为 .docx
