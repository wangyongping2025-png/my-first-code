# 📖 朗读文档小软件

上传 Word（.docx）、TXT 或 Markdown（.md）文档，选择喜欢的普通话声音，
用自然流畅、富有感情的声音为你朗读。

语音引擎使用微软 Edge 的神经网络语音（edge-tts），**完全免费**，
朗读效果接近真人，不是机械音。

## 功能

- 支持 .docx、.txt、.md 三种文档格式（TXT 自动识别 UTF-8 / GBK 编码）
- **6 种普通话声音**：女声晓晓、晓伊；男声云希、云健、云扬（新闻播报）、云夏（讲故事）
- 可调语速（慢 50% 到 快 50%）
- **长文档自动分段朗读**：一段一段读，衔接顺畅，不用一次等很久
- **暂停 / 继续 / 停止**：随时控制朗读
- **进度显示**：右下方列出所有段落，正在读的那段会高亮，点任意一段可以从那里开始听
- 上传后可以先预览、修改文字，再朗读

## 怎么运行（macOS）

### 最简单：双击启动

直接双击文件夹里的 **「启动朗读软件.command」**，它会自动装好组件、
启动软件并打开浏览器。

> 第一次双击如果系统提示「无法打开」，**右键点它 → 选「打开」→ 再点「打开」** 即可
> （这是 macOS 对下载文件的安全提示，只需确认一次）。

### 或者：用终端

按 `Command + 空格` 搜索「终端」打开，进入这个文件夹后执行：

```bash
pip3 install -r requirements.txt
python3 app.py
```

看到提示后，浏览器打开 http://127.0.0.1:5000 即可使用。

## 用完之后

回到终端窗口，按 `Control + C` 就能关闭软件。

## 注意

- 朗读时需要联网（语音是微软的在线服务生成的）
- 文档越长，第一段生成语音会等几秒，之后会自动提前准备下一段，听起来更连贯
- 老版 Word 的 .doc 格式不支持，请先用 Word 另存为 .docx
